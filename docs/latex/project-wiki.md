## Overview

eNow2 is a comprehensive telehealth platform that enables virtual care delivery through video calls, chat, and appointment management. This documentation is organized by user role for easy navigation and targeted information.

### Supported Languages

- English (Default)
- Spanish

Click on a role below to view its detailed documentation:

### Combined Roles

The platform also supports users with multiple role permissions:

- Provider-Admin
- Provider-Coordinator
- Admin-Coordinator
- Provider-Admin-Coordinator
- Super Admin (elevated administrative access)

Users with combined roles will have access to all features from their assigned roles.

## Common Features Across All Roles

### Profile Menu

All roles have access to a profile menu in the top-right corner showing:

- User name and email
- **Account settings** - Manage profile, notifications, and preferences
- **Help** - Access support documentation
- **Privacy policy** - View privacy policy
- **Log out** - End session

### Account Settings

Account settings vary by role but typically include:

#### My Account Tab (All Roles)

- Profile photo upload/delete
- Profile details editing
- Application language selection
- Time zone configuration
- Account deletion option

#### Calendar Tab (Provider Only)

- Time zone settings
- Daily availability scheduling (Monday-Sunday)
- Working hours configuration
- Calendar integrations (connect external calendars)

#### Notifications Tab (All Roles)

- Notification methods (Email, SMS, In-app)
- Session reminders configuration
- Notification preferences

## Getting Started

### First-Time Login

1. Navigate to https://portal.qa-encounterservices.com
2. Enter your email address
3. Enter your password
4. Accept End User License Agreement (EULA) if prompted
5. Complete "Before We Get Started" onboarding if required

### Device ID Login (Device Role Only)

1. Navigate to https://portal.qa-encounterservices.com/login/device
2. Enter your numeric Device ID (e.g., "33333")
3. Click "Verify Device ID"
